How to use this project:
1. Load project
2. Create new razor component using template from material.io
3. Pack library into a NUPKG
4. Use Package Manager Console to manually add this package into any Blazor.Client project
Command: Install-Package <path to RazorComponents.MaterialDesignxxxx.NUPKG
